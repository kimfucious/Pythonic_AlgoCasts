Node Class API

Function Arguments Returns Directions Example
Node definition (Data, Node) Node Creates a class instance to represent a node. The node should have two data attributes, 'data' and 'next'. Accept both of these as arguments to the 'Node' __init__() method, then assign them to the instance as data attributes 'data' and 'next'. If 'next' is not provided to the __init__ method, then default its value to be 'None'. Lastly, Python does have built-in function called next(). This may confuse the sytax highlighter in your text editor, but it's safe to use next here. 
            n = Node('There')
            n.data # 'Hi'
            n.next # None
            n2 = Node('Hi', n)
            n.next # returns n

LinkedList Class API

Function Arguments Returns Directions Example
constructor - (LinkedList) Create a class to represent a linked list. When created, a linked list should have *no* head node associated with it. The LinkedList instance will have one data attribute, 'head', which is a reference to the first node of the linked list. By default 'head' should be 'None'. 
            l = LinkedList()
            l.head # None
insert_first (data) - Creates a new Node from argument 'data' and assigns the resulting node to the 'head' data attribute. Make sure to handle the case in which the linked list already has a node assigned to the 'head' data attribute. 
            l = LinkedList()
            l.insert_first('Hi There') # List has one node
size - (integer) Returns the number of nodes in the linked list. 
            l = LinkedList()
            l.insert_first('a')
            l.insert_first('b')
            l.insert_first('c')
            l.size() # returns 3
get_first - (Node) Returns the first node of the linked list. 
            l = LinkedList()
            l.insert_first('a')
            l.insert_first('b')
            l.get_first() # returns Node instance with data 'a'
get_last - (Node) Returns the last node of the linked list 
            l = LinkedList()
            l.insert_first('a')
            l.insert_first('b')
            l.get_last() # returns node with data 'a'
clear - - Empties the linked list of any nodes. 
            l = LinkedList()
            l.insert_first('a')
            l.insert_first('b')
            l.clear()
            l.size() # returns 0
remove_first - - Removes only the first node of the linked list. The list's head should now be the second element. 
            l = LinkedList()
            l.insert_first('a')
            l.insert_first('b')
            l.remove_first()
            l.get_first() # returns node with data 'a'
remove_last - - Removes the last node of the linked list 
            l = LinkedList()
            l.insert_first('a')
            l.insert_first('b')
            l.remove_last()
            l.size() # returns 1
            l.get_last() # returns node with data of 'b'
insert_last (Data) - Inserts a new node with provided data at the end of the chain 
            l = LinkedList()
            l.insert_first('a')
            l.insert_first('b')
            l.insert_last('c')
            l.get_last() # returns node with data 'C'
get_at (integer) (Node) Returns the node at the provided index 
            l = LinkedList()
            l.insert_first('a')
            l.insert_first('b')
            l.insert_first('c')
            l.get_at(1) # returns node with data 'b'
remove_at (integer) - Removes node at the provided index 
            l = LinkedList()
            l.insert_first('a')
            l.insert_first('b')
            l.insert_first('c')
            l.remove_at(1)
            l.get_at(1) # returns node with data 'a'
insert_at (Data, integer) - Create an insert a new node at provided index. If index is out of bounds, add the node to the end of the list. 
            l = LinkedList()
            l.insert_first('a')
            l.insert_first('b')
            l.insert_first('c')
            l.insert_at('Hi', 1)
            l.get_at(1) # returns node with data 'Hi'
for_each (function) - Calls the provided function with every node of the chain 
            l = LinkedList()
            l.insert_last(1)
            l.insert_last(2)
            l.insert_last(3)
            l.insert_last(4)

            def fn(node):
                node.data += 10
                
            l.for_each(fn)

            l.get_at(0) # Returns node with data '11'
for...of Loop - - Linked list should be compatible as the subject of a 'for...in' loop 
            l = LinkedList()
            l.insert_last(1)
            l.insert_last(2)
            l.insert_last(3)
            l.insert_last(4)

            for node in l:
                node.data += 10

            node.get_at(1) # returns node with data 11
